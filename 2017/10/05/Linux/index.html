In Linux ,
if you want to transmit files between server and server ,
you can use these instructions.

  1. 傳送檔案(單個)

    從A伺服器傳檔案到B伺服器內
    於A伺服器內，
    請到你要傳送的目錄或檔案下
    接著用ftp方式登入B伺服器

    登入B伺服器後，
    在B伺服器內，
    前往你想要傳送到的目的地檔案夾下 (Ex: /mnt/imdata1/Lee)

    指定port

    BTW. 若是sftp要傳送的伺服器port是有特定的話指令改為
